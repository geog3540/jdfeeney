James Feeney Project Portfolio

Choropleth Mapping

I created a choropleth map of Wisconsin's white population using Observable tools. I decided to use manual breaks for this project, and found very interesting that there is one county with a white population that is far in the minoirty to the Native population.

Proportional Symbol Mapping

I created proportional symbols for Wisconsin's Park and Ride locations for bus transit, symbolizing the number of spots. I was surpised at how many there were outside of the Madison and Milwaukee areas.

Isarithmic Mapping

This is Dr. Koylu's Isarithmic Map of Iowa, symbolizing yearly total precipitation. I plan to use some sort of an isarithmic map for my final project.

Bivariate Choropleth Mapping

This is a bivariate map of Minnesota, which used U.S. Census data. It visualized median household income and the percentage of the workforce taking public transit to work in Minnesota counties. I notice that the counties with a high class for both, they are around the Minneapolis area, while counties low for both are spread throughout all corners of the state.

Flow Mapping

I utilzied Dr. Koylu's data and tool to create a flow map of imports and exports of bananas throughout South America. I ended up using raw counts for this map, which is not entirely inapprpiate, since bananas are not exactly normalized by anything. However, it seems like using a ratio for at least one of the symbols could improve this map further.

Chicago Rail Accesibility StoryMap (Final Project)

I created visualizations of rail transit throughout the city of Chicago. I utilized tools in ArcGIS Network Analysis to display walking time to stations, and compared that to event venue locations, poplation density, housing value, and when structures were built..

Multiple Small Choropleths

I utilized a template in order to visualize the languages of Utahns. The map includes a hover over fature where hovering over one county will give you information about the same county for other variables. It also includes multiple classifications that a user can toggle between.
